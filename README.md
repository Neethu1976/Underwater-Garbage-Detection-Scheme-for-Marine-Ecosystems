🌊 Underwater Garbage Detection System for Marine Ecosystems

This project presents an optimized YOLOv8-Small based deep learning model for detecting underwater debris in challenging marine environments. To address issues like low visibility, color distortion, and class imbalance, advanced preprocessing and augmentation techniques are applied.

🚀 Key Features
CLAHE-based Contrast Enhancement to improve image clarity in low-visibility underwater conditions
YOLOv8-Small Model for lightweight, fast, and real-time object detection
Mosaic Data Augmentation to handle class imbalance and improve generalization
Trained on a 15-class underwater debris dataset
Optimized for accuracy + efficiency, making it suitable for real-time applications
🧠 Tech Stack
Python
YOLOv8 (Ultralytics)
OpenCV
NumPy / Pandas
📊 Outcome

The model achieves strong detection performance while remaining computationally efficient, making it ideal for deployment in marine monitoring systems and environmental conservation efforts.

🌍 Impact

This system can assist in:

Monitoring ocean pollution
Supporting marine conservation initiatives
Enabling automated waste detection in underwater environments
