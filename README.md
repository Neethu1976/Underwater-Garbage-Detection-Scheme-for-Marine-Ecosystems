🌊 Underwater Garbage Detection System for Marine Ecosystems

This project proposes a robust and efficient deep learning-based approach for detecting underwater debris using an optimized YOLOv8-Small model. It is specifically designed to address the challenges of underwater imaging, such as low visibility, color distortion, and uneven lighting conditions.

📌 Overview

The system integrates advanced image preprocessing and data augmentation techniques to enhance detection performance in complex marine environments. Contrast Limited Adaptive Histogram Equalization (CLAHE) is applied to improve image quality, while Mosaic augmentation is used to mitigate class imbalance within the dataset.

⚙️ Key Features
Lightweight and high-performance YOLOv8-Small architecture for real-time detection
CLAHE-based preprocessing to enhance contrast and visibility in underwater images
Mosaic data augmentation to improve model generalization and handle class imbalance
Trained on a multi-class (15 categories) underwater debris dataset
Optimized for accuracy, speed, and deployment efficiency
